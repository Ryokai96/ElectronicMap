# C语言电子地图管理系统
本项目目的为C语言数据结构巩固，C语言文件操作

使用工具：
- 系统：windows xp sp3
- 编译器：MinGW4.8
- IDE：QT Creator



项目功能：

- 从DATA文件夹中的二进制文件GTBL.dat读取地图数据，对读取出来的数据进行排序、查找、删除、插入、更新操作
- 读取方式可选择
- 排序方式可选择，将排序后的数据输出到SortGTBL.dat中
- 查找的方式可选择，查找结果少于5条则输出到控制台，否则输出到文件searchresult.txt中
- 更新功能即将SortGTBL.dat替换GTBL.dat
